# Code of Conduct

## Our Standards

We expect all contributors to interact professionally and respectfully. This project is a technical collaboration focused on building quality software.

**Expected behavior:**

- Be respectful and constructive in all communications
- Focus on technical merit and problem-solving
- Welcome newcomers and help them get started
- Accept constructive criticism gracefully
- Respect differing viewpoints and experiences

**Unacceptable behavior:**

- Personal attacks, insults, or harassment
- Spam, trolling, or off-topic disruptions
- Publishing others' private information without permission
- Any conduct that would be inappropriate in a professional setting

## Enforcement

Project maintainers have the right and responsibility to remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct. Maintainers may ban temporarily or permanently any contributor for behaviors they deem inappropriate, threatening, or harmful.

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, discussions, and any other communication channels associated with this project.

## Contact

Instances of unacceptable behavior may be reported by opening an issue or contacting the project maintainer at https://github.com/georgialexandrov.
